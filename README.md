# LifeOS Frontend

Frontend-приложение LifeOS на React, TypeScript и Vite.

## Запуск проекта

Для запуска проекта требуется установленный Docker.

Запустите проект командой:

```bash
docker compose up --build
```

После запуска frontend будет доступен по адресу:

```text
http://localhost:5173
```

## Остановка проекта

```bash
docker compose down
```
